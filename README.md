# Java Test Programs - Poojitha

This repository contains **4 Java programs** for a coding test. Each program is implemented in **Java** and demonstrates basic programming concepts such as classes, loops, and arrays.

---

## **Programs**

### **Program-1: Calculator using Class**
- **Description:** Performs basic arithmetic operations (Addition, Subtraction, Multiplication, Division) using a `Calculator` class.
- **Input:**  
  - `a` → double  
  - `b` → double  
  - `type of operation` → string (`add`, `subtract`, `multiply`, `divide`)  
- **Output:** Result of the selected operation.
- **File:** `Program-1.java`

---

### **Program-2: Odd Number Series**
- **Description:** Prints odd numbers from 1 up to the nth odd number, based on user input.
- **Input:** Single integer `n`
- **Output:** Series of odd numbers separated by commas.
- **File:** `Program-2.java`

---

### **Program-3: Odd Number Series with Repetition Pattern**
- **Description:** Prints a series of odd numbers with a repetition pattern based on user input.
- **Input:** Single integer `n`
- **Output:** Series of odd numbers for each input step, following a pattern.
- **File:** `Program-3.java`

---

### **Program-4: Count Multiples in Array**
- **Description:** Counts how many numbers in an array are multiples of numbers from 1 to 9.
- **Input:** Array of integers  
- **Output:** Map showing counts of multiples for numbers 1 to 9.  
- **File:** `Program-4.java`
